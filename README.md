### Introduction

`log4vue` is a customizable logger for [Vue.js](http://vuejs.org).

### Development Setup

``` bash
# install deps
npm install

# build dist files
npm run build

# serve examples at localhost:8080
npm run dev

# lint & run all tests
npm test

# serve docs at localhost:4000 (requires global gitbook-cli)
npm run docs
```
